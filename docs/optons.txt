-l --level How deep to traverse the file tree (default 3)
-d --directory : show directories only
-a --annotated: show annotated files only
-n --non-annotated: show not annotated files only
-e --excludes : a pattern to ecludes (can be used multiple times) as in
.gitignore patterns
-m --modified : show only files modified (by git)
-s --staged : show only staged files (by git)
-h --hidden : show hidden files (those starting with .)
--ignores-file-no Do not use a ignores file (by default .gitignore is honored)
--ignore-file-path The path to ignore files (by default .gitignore)
